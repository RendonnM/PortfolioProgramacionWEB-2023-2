HTML

Respuesta

Lenguaje de marcado utilizado para crear la estructura y contenido de las páginas web.

CSS

Respuesta

Lenguaje de estilo utilizado para definir la apariencia y diseño de las páginas web.

JavaScript

Respuesta

Lenguaje de programación utilizado para crear interactividad y funcionalidad en las páginas web.

HTTP

Respuesta

Protocolo de comunicación utilizado para transferir datos entre un cliente web y un servidor.

URL

Respuesta

Dirección única que identifica la ubicación de un recurso en la web.

DOM

Respuesta

Representación en forma de árbol de la estructura de un documento HTML que permite acceder y manipular elementos web.

API

Respuesta

Conjunto de reglas y protocolos que permiten que diferentes aplicaciones se comuniquen entre sí.

Responsive Design

Respuesta

Enfoque de diseño web que garantiza que las páginas se vean bien y funcionen correctamente en diferentes dispositivos y tamaños de pantalla.

CMS

Respuesta

Sistema que facilita la creación y gestión de contenido en sitios web, como WordPress o Drupal.

SEO

Respuesta

Conjunto de prácticas para mejorar la visibilidad de un sitio web en los motores de búsqueda.

Backend

Respuesta

La parte de un sitio web que se encarga del procesamiento y almacenamiento de datos, generalmente oculta al usuario.

Frontend

Respuesta

La parte de un sitio web que se muestra y con la que interactúa el usuario, generalmente construida con HTML, CSS y JavaScript.

Framework

Respuesta

Conjunto de herramientas y bibliotecas predefinidas que simplifican el desarrollo web, como Angular, React o Vue.

Cookie

Respuesta

Pequeño archivo de texto que un sitio web almacena en la computadora del usuario para mantener información sobre su actividad en línea.

HTTPS

Respuesta

Versión segura de HTTP que utiliza cifrado para proteger la comunicación entre el navegador y el servidor.

AJAX

Respuesta

Tecnología que permite la carga de contenido en una página web sin necesidad de recargarla por completo.

Hosting

Respuesta

Servicio que proporciona espacio en servidores web para almacenar y servir sitios web en línea.

Git

Respuesta

Sistema de control de versiones ampliamente utilizado para el seguimiento de cambios en el código fuente.

SaaS

Respuesta

Modelo de distribución de software en el que las aplicaciones se ofrecen como un servicio a través de la web.

MVC

Respuesta

Patrón de diseño de software que divide una aplicación en tres componentes principales: Modelo, Vista y Controlador.
